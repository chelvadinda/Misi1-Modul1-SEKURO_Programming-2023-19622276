# Bekerja Dengan Git

~~~
$ git init 
untuk menginisiasi repo git
$ git add <file(s)> 
untuk menambah file
$ git status
untuk mengetahui status repo
$ git commit
untuk melakukan commit
$ git config
untuk memasukkan konfigurasi
$ git branch
untuk membuat branch
$ git help
untuk menampilkan daftar perintah
~~~
